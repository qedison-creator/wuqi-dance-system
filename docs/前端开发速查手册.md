# 舞栖舞蹈社 - 前端开发速查手册

## 🚀 快速开始

### 启动后端服务
```bash
cd wuqi-backend
npm install
npm run seed      # 初始化数据库（仅首次）
npm run dev       # 启动服务
```

### 启动小程序
1. 打开微信开发者工具
2. 导入项目（选择 `wuqi-member` 或 `wuqi-admin`）
3. 点击"编译"按钮

## 📦 工具库使用

### API 调用（会员端）
```javascript
const api = require('../../utils/api');

// 首页数据
api.home.getBanners({ store_id }).then(res => {
  console.log(res.data);
});

// 获取我的套餐
api.packages.getMy().then(res => {
  console.log(res.data);
});

// 预约课程
api.bookings.create({ schedule_id: 'xxx' }).then(res => {
  wx.showToast({ title: '预约成功' });
});
```

### API 调用（管理端）
```javascript
const api = require('../../utils/api');

// 获取会员列表
api.members.getList({ status: 'registered', page: 1 }).then(res => {
  console.log(res.data.list);
});

// 审核会员
api.members.review(memberId, { action: 'approve' }).then(() => {
  wx.showToast({ title: '审核成功' });
});

// 签到
api.bookings.checkIn({ schedule_id, user_id }).then(() => {
  wx.showToast({ title: '签到成功' });
});
```

### 工具函数使用
```javascript
const helpers = require('../../utils/helpers');

// 格式化日期
helpers.formatDate(new Date(), 'YYYY-MM-DD');
helpers.formatDateTime(new Date());

// Toast 提示
helpers.showToast('操作成功');
helpers.showLoading();
helpers.hideLoading();

// 弹窗
helpers.showModal('提示', '确认要删除吗？').then(() => {
  // 用户点击确认
}).catch(() => {
  // 用户点击取消
});

// 格式化金额（管理端）
helpers.formatMoney(1000); // ¥1,000.00
```

## 🔧 常见问题

### 1. 小程序无法连接后端
- 确认后端服务已启动
- 微信开发者工具勾选"不校验合法域名"
- 检查 `app.js` 中的 `baseUrl` 配置

### 2. 登录失败
- 确认数据库已初始化（`npm run seed`）
- 检查用户名密码是否正确

### 3. 上传图片失败
- 检查 `uploads` 文件夹是否存在
- 确认有写权限

## 📝 开发流程

### 新功能开发
1. 在后端实现 API
2. 在 `utils/api.js` 中添加 API 方法
3. 在页面中调用 API
4. 测试功能

### 代码规范
- 使用 ES6+ 语法
- 函数命名使用驼峰式
- 添加必要的注释
- 错误处理要完善
