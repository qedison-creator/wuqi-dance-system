## 🔧 环境准备

### 1. 安装必要软件
- **Node.js** (v14+): https://nodejs.org)
- **MongoDB** (v4.4+): https://www.mongodb.com/try/download/community
- **微信开发者工具**: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 启动 MongoDB

#### Windows
```cmd
# 方法1: 启动 MongoDB 服务
net start MongoDB

# 方法2: 如果没有安装服务，直接运行
mongod
```

#### macOS/Linux
```bash
# macOS (Homebrew安装后
brew services start mongodb-community
# 或
mongod
```

## 🚀 后端服务启动

### Step 1: 进入后端目录
```bash
cd wuqi-backend
```

### Step 2: 安装依赖
```bash
npm install
```

### Step 3: 初始化数据库
```bash
npm run seed
```

### Step 4: 启动服务
```bash
npm run dev
```

服务将在 `http://localhost:3000` 启动

### 🎯 或者直接使用快速启动脚本

Windows:
```cmd
quick-start.bat
```

macOS/Linux:
```bash
chmod +x quick-start.sh
./quick-start.sh
```

## 📱 小程序开发

### 会员端 (wuqi-member)
1. 打开微信开发者工具
2. 导入项目：选择 wuqi-member 目录
3. 设置 AppID（测试可用测试号）

### 管理端 (wuqi-admin)
1. 打开微信开发者工具
2. 导入项目：选择 wuqi-admin 目录
3. 登录管理账号

### 小程序配置
- API 地址已配置为 `http://localhost:3000/api/v1`
- 在微信开发者工具中，勾选"不校验合法域名"

## ⚠️ 常见问题

### 端口被占用
```env
# 修改 .env 中的 PORT 端口号
PORT=3001
```

### MongoDB连接失败
检查 MongoDB 确保已启动 MongoDB 服务
```env
# 检查 .env 中配置的 MongoDB 连接字符串
MONGODB_URI=mongodb://localhost:27017/wuqi_dance
```

### 小程序无法连接后端
1. 确认后端服务已启动
2. 微信开发者工具中勾选"不校验合法域名"
